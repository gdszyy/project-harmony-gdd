## 1. 概述

本文档旨在对《Project Harmony》游戏设计文档（GDD）中已有的角色概念进行深化与扩展，为主角及敌人提供一套更丰富、具体且富有表现力的视觉设计方案。所有提案严格遵循 GDD 中确立的“抽象矢量主义 vs. 数字衰变”双极美学体系，并致力于将角色的视觉表现与其在游戏中的音乐隐喻和玩法机制进行深度绑定，以实现“通感”（Synesthesia）的核心美学追求。

## 2. 主角设计提案：化身“谐振调式”

为解决当前主角纯几何形态“角色感”不足的问题，并与 GDD 中“基于调性与音阶的职业系统”构想相结合，我们提议将主角设计为不同音乐调式（Mode）的化身。每个调式都将拥有独特的几何形态、色彩倾向和动态视觉，直观地反映其音乐性格与玩法定位，从而为玩家提供多样化的游戏体验和丰富的重玩价值。

### 2.1. 核心理念：形态服务于功能

玩家选择的不同“职业”即是不同的音乐调式化身。这种设计不仅在叙事上强化了“玩家即音乐”的概念，更在玩法上通过视觉语言直观引导玩家理解不同职业的优劣势。

### 2.2. 调式化身设计

我们设计了四种具有代表性的调式化身，以展示该系统的潜力。所有化身将共享一套通用的表现层逻辑，即结合“无形指挥家”的动态概念，根据法术类型做出不同的施法动作（如单指前点、双手画圆、高举等），以增强表现力。

| 调式化身 | 音乐性格 | 玩法定位 | 视觉设计 |
| :--- | :--- | :--- | :--- |
| **爱奥尼亚式 (Ionian)** | 明亮、稳定、和谐 | **均衡型**：无明显短板，适合新手。 | **形态**：维持 GDD 中的基准形态——三层同心完美圆形光环，围绕一个正十二面体核心，旋转平滑稳定。<br>**色彩**：主色为“谐振青”，核心为“晶体白”。<br>**动态**：光环与BPM精准同步旋转，施法时向外产生柔和的涟漪。 |
| **洛克里亚式 (Locrian)** | 紧张、不稳定、不和谐 | **高风险型**：依赖不和谐机制造成巨大伤害，但自身更脆弱。 | **形态**：光环为**不闭合的弧线**，边缘有轻微的“数字衰变”毛刺。三道弧线以非对称的轨迹和不稳定的速度旋转。<br>**色彩**：主色为“腐蚀紫”，核心在紫与红之间脉动。<br>**动态**：施放不和谐法术时，弧线会剧烈抖动并产生尖锐的视觉“毛刺”。 |
| **吕底亚式 (Lydian)** | 极度明亮、开放、梦幻 | **范围控制型**：法术的范围（SIZE）和持续时间（DUR）有加成。 | **形态**：光环形态近似**星云或星环**，边缘模糊并散发大量微小粒子，视觉范围更广。<br>**色彩**：主色为“圣光金”与“和弦蓝”的混合，粒子带有彩虹色泽。<br>**动态**：光环缓慢旋转，但其上的粒子高速流动。施法时产生向外扩散的星尘波纹。 |
| **弗里几亚式 (Phrygian)** | 异域、尖锐、攻击性 | **进攻型**：法术的伤害（DMG）和飞行速度（SPD）有加成。 | **形态**：核心为**尖锐的晶体簇**。光环变为锋利的刀锋状，边缘锐利，并有缓慢的开合动作。<br>**色彩**：主色为“错误红”与“霓虹粉”的结合，核心为炽白色。<br>**动态**：光环旋转速度快，施法时会向发射方向进行一次快速的“刺击”动作。 |

## 3. 敌人设计深化

本章节在 GDD 已有的敌人设计基础上，进一步深化其视觉细节，强化“音乐错误具象化”的理念，并详细阐述其在“频谱相位系统”下的不同形态。

### 3.1. 基础敌人视觉深化

#### Static (底噪) - “无序的涌动”

- **音乐隐喻**：白噪声，无处不在的背景干扰。
- **视觉深化**：其不规则多面体的表面布满了微小、快速变化的**数据蚊（Datamosh）**伪影，轮廓在低帧率移动中不断轻微“抽搐”和变形。
- **相位形态**：
    - **高通**：缩小为光点后，运动轨迹变为**布朗运动**式的随机折线，身后拖拽短暂的洋红色光纤。
    - **低通**：膨胀为减速雾团后，内部充满了缓慢旋转的、更深色的**像素块（Pixelation）**。

#### Silence (寂静) - “吞噬声音的奇点”

- **音乐隐喻**：休止符/黑洞，声音的缺失。
- **视觉深化**：其核心是能够吸收光线的“绝对黑体”，周围的“静音光环”被设计成一种**空间扭曲透镜**，使光环内的背景发生可见的弯曲。
- **相位形态**：
    - **高通**：暴露的核心是一个快速旋转的、由白色光构成的**多维晶体**，内部折射出混乱的彩虹色光芒。
    - **低通**：扩大的光环扭曲效果更强，并会周期性地向外喷射出墨水般的黑色粒子。

#### Screech (尖啸) - “失控的反馈”

- **音乐隐喻**：麦克风反馈音，刺耳的自激振荡。
- **视觉深化**：三角锥体表面周期性地出现**扫描线（Scanline）**滚动效果。准备冲刺时，其尖端会变得极度明亮并发出高频振动的波纹。
- **相位形态**：
    - **高通**：变为电弧后，瞬移时会在原地留下**残影**，并在目标位置由一团电弧火花中“重组”而成。
    - **低通**：被压制为静止光球后，表面会浮现出不断变化的、混乱的**示波器声波波形**图案。

#### Pulse (脉冲) - “错误的节拍器”

- **音乐隐喻**：错误的节拍器，在不该出现的时点爆发。
- **视觉深化**：正八面体的每个面都像一个独立的**LED面板**，在蓄力时会依次亮起，形成倒计时效果。
- **相位形态**：
    - **高通**：瞬发激光由一系列快速闪烁的**矩形光段**组成，更具“数字感”。
    - **低通**：全屏震荡波是从屏幕中心爆开的、可见的**矩形像素网格**，如同一次全屏的“分辨率骤降”攻击。

#### Wall (音墙) - “压缩的极限”

- **音乐隐喻**：砖墙限制器，将动态范围压平。
- **视觉深化**：其表面的EQ频谱**实时响应背景音乐**。护盾是一个独立的、带有六边形蜂巢纹理的半透明能量场。
- **相位形态**：
    - **高通**：变为线框骨架后，线条是**断续闪烁的**，移动时在身后留下线框残影。
    - **低通**：变为不可摧毁的巨石后，表面布满**裂缝**，缝中有红色光芒缓慢流动。

### 3.2. 章节特有敌人视觉深化

为使章节特有敌人更具特色，并与章节主题紧密关联，我们提出以下视觉深化方案：

| 章节 | 敌人 | 视觉深化 |
| :--- | :--- | :--- |
| **第二章** | **唱诗班 (Choir)** | 编队中的 Static 由可见的暗红色**能量链**连接，领头的 Static 体型稍大且核心更亮，如同一个统一的意志。 |
| **第三章** | **对位爬虫 (Crawler)** | 机械蜘蛛的身体是布满黄铜齿轮的**羽管键琴**结构，腿是长短不一的音管，炮塔是旋转的**管风琴**音管簇。 |
| **第四章** | **小步舞曲舞者 (Dancers)** | 成对的抽象剪影，形态为**洛可可风格的华丽烛台**。它们关于中心点镜像移动，旋转时变为一团无法锁定的金色光屑旋风。 |
| **第五章** | **狂怒精魂 (Fury Spirit)** | 不定形的闪电元素，核心是一个**跳动的心脏状闪电球**。其形态会随BGM情绪激烈程度而变化，从柔和电光到狂暴的闪电触须。 |
| **第六章** | **摇摆贝斯 (Walking Bass)** | 低音提琴的轮廓由**霓虹灯管**构成，颜色随布鲁斯音阶流动。其行走轨迹是一个个短暂存在的发光**音符符号**。 |
| **第七章** | **比特破碎虫 (Worm)** | 身体由**不断变化的、不同分辨率的像素块**构成。它会“感染”其经过的地面，使其暂时也变成低分辨率的像素块。 |

## 4. 结论

本提案在 GDD 现有框架的基础上，为主角和敌人提供了更为详尽和系统化的视觉设计方向。主角的“谐振调式”系统将角色多样性与核心音乐机制深度绑定，而敌人的视觉深化则让“数字衰变”的美学理念和“音乐错误”的隐喻更加生动和具体。这些设计共同服务于游戏“通感”的终极目标，旨在为玩家创造一场沉浸式的、在战斗中进行音乐创作的独特视听盛宴。
