## 0. 设计哲学：场景即是乐器

本文档旨在为《Project Harmony》的七个章节设计独特的场景地图机制与美术风格，作为对核心GDD [1]、关卡设计 [2] 和美术方向 [3] 的补充与深化。我们的核心设计哲学是**“场景即是乐器”**——地图不仅是战斗发生的舞台，更是游戏核心机制的延伸，是玩家可以“演奏”的互动环境。

每一个场景小机制都将紧密围绕该章节的核心教学目标和音乐主题，通过环境互动，让玩家更直观、更深刻地理解从节奏、和弦到相位切换的演进过程。美术风格则致力于将抽象的音乐史概念转化为强烈的视觉体验，让玩家“看”到音乐的情感与结构。

## 第一章：律动尊者·毕达哥拉斯

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 秩序 vs. 混沌 |
| **场景基调** | 纯粹、神圣、精确的数学与几何之美 |
| **美术风格** | **“光之神殿”**：极简主义、几何构成、高对比度。灵感源于古希腊神庙的纯粹线条与《Antichamber》等游戏的抽象空间感。场景主体为一个由纯白发光线条构成的圆形殿堂，地面是由脉冲光构成的网格。色调以**白、金、黑**为主。白色代表秩序与和谐，金色用于“完美卡拍”等正向反馈，黑色则代表“噪音”与混沌的敌人。视觉元素强调清晰的几何形状（圆形、三角形、直线），避免任何复杂的纹理和装饰。 |
| **地图小机制** | **“克拉尼图形安全区” (Chladni Plate Safe Zones)**：该机制是对Boss战核心机制的预演和教学。在特定波次中，地面原本均匀的脉冲网格会开始“振动”，并根据特定的频率形成**克拉尼图形**。发光的线条区域会变成对玩家造成持续微弱伤害的“失谐区”，而线条之间的静止区域则是“和谐区”（安全区）。这迫使玩家必须跟随地面图形的变化而移动，将“走位”与章节核心的“节奏感”和“秩序感”紧密结合。随着关卡推进，图形会从简单的同心圆演变为复杂的花瓣状，潜移默化地训练玩家在Boss战中的走位能力。 |

## 第二章：圣咏宗师·圭多

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 单声 vs. 复调萌芽 |
| **场景基调** | 庄严、幽深、充满混响的哥特式教堂 |
| **美术风格** | **“哥特回响”**：高耸的垂直空间感、巨大的花窗玻璃、悠长的阴影。灵感来自真实世界的哥特式大教堂和《暗黑破坏神》系列的场景氛围。主色调为**岩石灰、深蓝、暗紫**，花窗玻璃会投下彩色的光斑，在地面形成流动的光池。场景的“混响”感将通过视觉呈现：玩家的每次和弦攻击都会产生一圈短暂、微弱的“视觉回声”，以光波的形式扩散出去，强化教堂的空灵感。 |
| **地图小机制** | **“回声圣坛” (Echoing Altars)**：地图中散布着几个小型的祈祷圣坛。当玩家在圣坛附近（一定范围内）释放一个**和弦**时，圣坛会被“激活”并发出微光，将这个和弦“记录”下来。在短暂延迟（例如2秒）后，圣坛会向周围“咏唱”出一个弱化版的同类和弦法术（例如，玩家释放大三和弦，圣坛就回响一个范围较小的大三和弦冲击波）。这个机制有多重意义：
1.  **机制教学**：它在场景层面创造了最朴素的“复调”——玩家的声部与圣坛的声部构成了简单的Call and Response，直观地呼应了本章“单声vs复调”的主题。
2.  **策略深度**：玩家可以将圣坛作为战术性的“延迟陷阱”或“自动炮台”使用，通过预判敌人走位来提前“布置”和弦，增加了战斗的策略维度。
3.  **风险管理**：激活圣坛需要玩家靠近，这在某些情况下会带来风险，玩家需要在“安全输出”和“利用场景机制”之间做出权衡。 |

## 第三章：大构建师·巴赫

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 复调 vs. 主调 |
| **场景基调** | 由管风琴管道与精密齿轮构成的机械宇宙 |
| **美术风格** | **“巴洛克机械城”**：灵感源于钟表内部的精密结构、蒸汽朋克美学以及管风琴的宏伟构造。整个场景如同一个巨大而神圣的音乐盒内部，充满了缓慢而精确运动的机械部件。主色调为**黄铜、青铜、钢铁和深色木材**。视觉元素的核心是巨大的旋转齿轮、同步起落的活塞，以及在背景中随着玩家和弦变化而依次亮起的发光管风琴管道。背景是一个由无数机械部件交错构成的复杂网络，营造出一种庄严、精密且充满逻辑性的美感。 |
| **地图小机制** | **“对位齿轮平台” (Counterpoint Gear Platforms)**：战斗场景并非一个平面，而是由数个缓慢旋转的巨型齿轮平台构成。这些平台相互啮合，其转速与游戏的BPM严格同步，将“多声部”这一音乐概念转化为立体的战场环境。
1.  **立体战场**：这种设计创造了高低错落的立体战场，迫使玩家在平台间跳跃移动，以攻击不同位置的敌人或躲避攻击。这种垂直性直接呼应了复调音乐中多条旋律线并行的概念。
2.  **节奏化移动**：平台的旋转是富有节奏且可预测的。玩家必须将自己的移动和跳跃与机械的“节拍”相协调，这强化了本章对复杂节奏结构的训练目标。
3.  **和弦进行交互**：特定的平台可以通过**和弦进行**进行操控。例如，一次成功的 **D→T (属功能→主功能)** 进行，可能会使玩家所在的平台短暂锁定，创造一个稳定的输出环境；反之，一次 **T→D (主功能→属功能)** 进行，则可能加速邻近载有敌人的平台，使其失衡或掉入场景陷阱。这将本章的核心机制“和弦进行”与环境控制能力直接挂钩，让玩家“演奏”整个场景。 |

## 第四章：古典完形·莫扎特

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 结构 vs. 即兴 |
| **场景基调** | 明亮、对称、优雅的洛可可风格宫廷舞厅 |
| **美术风格** | **“洛可可水晶厅”**：明亮、通透、极尽奢华。灵感来自凡尔赛宫的镜厅和古典主义时期的精致美学。主色调采用**粉彩（如淡粉、天蓝）、乳白和大量的金箔**。环境一尘不染，绝对对称，充满了镜面、水晶吊灯和抛光大理石等反射性表面。场景的完美对称性是核心视觉语言，强调古典主义对形式美和结构完整的追求。 |
| **地图小机制** | **“镜像对称轴” (Mirrored Symmetry Axis)**：一条闪烁着微光的能量线将舞厅完美地一分为二。这条看不见摸不着的轴线是本章地图机制的核心，具化了“对称”与“结构”的主题。
1.  **弹道反射**：玩家发射的**单音**弹体（非复杂和弦）在接触到这条轴线时，会以完美的对称角度被反射。这允许玩家利用几何学进行巧妙的“反弹射击”，攻击到被障碍物遮挡或位于意外角度的敌人，奖励玩家对空间结构和形式的理解，呼应了古典主义作曲的精确性。
2.  **敌人对称生成**：特定的敌人波次会严格沿着对称轴在两侧同时生成。一些较为脆弱的敌人甚至会在穿过轴线时被“镜像”，在另一侧复制出一个相同的敌人，迫使玩家必须同时兼顾两个战线，深刻体会“平衡”与“对称”的压力。
3.  **“奏鸣曲式”区域**：地图被轴线隐性地划分为“主部主题区”和“副部主题区”。在特定波次中，一侧区域可能会附加增益效果（如“展开部”：伤害提升），而另一侧则附加减益效果（如“再现部”：敌人速度加快），迫使玩家在两个区域间穿梭，如同演奏一首奏鸣曲的不同乐章。 |

## 第五章：狂想者·贝多芬

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 形式 vs. 情感 |
| **场景基调** | 风暴肆虐的荒野，崩坏的废墟，激烈、崇高且充满情感的自然力量 |
| **美术风格** | **“浪漫主义风暴”**：灵感源于卡斯帕·大卫·弗里德里希等浪漫主义风景画，强调自然的崇高与戏剧性。场景设定在狂风暴雨的荒野之巅，背景是翻滚的乌云和划破天际的闪电，地面是风化的岩石与哥特式废墟的残垣断壁。色调以**深灰、靛蓝、暗紫**为主，闪电的瞬间亮光会投下锐利、动态的阴影，营造出强烈的情感冲击力。环境本身就是动态的，风吹动着废墟上的破布，雨水在地面汇集成反光的水洼。 |
| **地图小机制** | **“命运雷暴” (Tempest of Fate)**：将本章的“动态BPM”和“不和谐度”机制与动态天气系统深度绑定。
1.  **节奏性落雷**：闪电不再是随机的背景，而是成为核心的场景威胁。落雷会严格遵循本章Boss的“命运动机”节奏（短-短-短-长）劈向地面，在落点产生持续伤害的电场。这既是环境威胁，也是玩家掌握Boss核心节奏的“环境节拍器”。
2.  **风眼与激流**：场景中会周期性出现“风眼”，即本章教学波次中引入的动态BPM区域。而在风眼之外，是狂风大作的“激流区”，会持续地、缓慢地将玩家和敌人向特定方向推动，影响走位和弹道，强化了场景的混乱与不可预测性。
3.  **不和谐共鸣**：当玩家的“不和谐度”超过一定阈值时，会与风暴产生“共鸣”。天空的闪电会变得更加频繁，落雷会优先劈向玩家所在位置，水洼被闪电击中后会变成危险的导电区域。这使得管理不和谐度不仅是维持自身战斗力的需要，更是避免场景“愤怒”的关键，将内在的情感状态（不和谐）与外在的环境（风暴）联系起来。 |

## 第六章：摇摆公爵·艾灵顿

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 谱写 vs. 即兴 |
| **场景基调** | 烟雾缭绕、霓虹闪烁的1930年代爵士俱乐部，慵懒、摇摆而又暗藏机锋 |
| **美术风格** | **“装饰风艺术俱乐部” (Art Deco Club)**：融合了装饰风艺术（Art Deco）的几何线条感与黑色电影（Film Noir）的光影运用。场景是一个高档的地下爵士俱乐部，核心是一个被吧台和卡座环绕的圆形舞池。视觉元素包括黄铜栏杆、天鹅绒幕布、以及闪烁的霓虹灯招牌（拼出“SWING”、“CALL”、“RESPONSE”等字样）。主色调为**暗红、宝蓝、墨绿与金色**，霓虹灯在烟雾弥漫的空气中投下柔和而暧昧的光线，形成强烈的光影对比。 |
| **地图小机制** | **“即兴聚光灯” (Improvisational Spotlight)**：将爵士乐中的“独奏”与“互动”概念转化为场景机制。
1.  **声部聚光灯**：一束聚光灯会在战斗中按照“摇摆节奏”在舞台上移动。当聚光灯锁定某个敌人（尤其是“即兴独奏者”）时，该敌人会进入“独奏”状态，攻击欲望和技能复杂度大幅提升，成为高威胁目标。这模拟了乐队中某个声部的即兴华彩乐段。
2.  **抢夺聚光灯**：如果玩家在聚光灯区域内完成一次高难度操作（例如一次成功的扩展和弦释放，或一次完美的相位切换反击），就能“抢夺”聚光灯。在聚光灯下，玩家会获得显著的增益（如伤害提升、冷却缩减），成为舞台的焦点。这为玩家提供了高风险高回报的玩法选择。
3.  **相位指示**：聚光灯的颜色会随着玩家切换的“频谱相位”而改变（例如高通相位为蓝色，低通相位为红色，全频为白色）。这为本章核心教学机制“相位切换”提供了最直观、最中心的视觉反馈，让玩家在关注战斗的同时，能时刻明确自己所处的“音乐频段”。 |

## 第七章：合成主脑·噪音

| 项目 | 设计方向 |
| :--- | :--- |
| **核心冲突** | 音乐 vs. 噪音 |
| **场景基调** | 抽象、破碎、不断变化的数字空间，是音乐宇宙崩溃的边缘 |
| **美术风格** | **“故障艺术空间” (Glitch Art Space)**：灵感来自示波器波形、频谱分析图以及故障艺术（Glitch Art）。这是一个完全非写实的抽象空间，背景是流动的数字瀑布和破碎的数据块。地面和墙壁由跳动的频谱图和失真的像素格构成。主色调是**数字绿、赛博蓝、品红和纯黑**，伴随着高饱和度的像素色块随机闪烁。所有视觉元素都在不断地轻微抖动、错位和变形，营造出一种不稳定、濒临崩溃的末日感。 |
| **地图小机制** | **“频谱污染” (Spectrum Corruption)**：这是对最终Boss战“波形战争”机制的预演，将“频谱相位”与场景本身融为一体。
1.  **相位专属区域**：地图会被划分为几个不规则的、边界不断变化的区域。每个区域都属于一个特定的频谱相位（高通、低通、带通等）。例如，在“高通区”，只有高频系的敌人和弹幕是可见且有效的，而在“低通区”则反之。玩家必须不断切换自身相位以匹配所在区域的属性，否则将处于“失明”或“哑火”状态。
2.  **相位偏移疲劳源**：长时间停留在某个单一的相位区域，会导致该区域被“污染”，视觉上的“故障”效果会加剧，并加速玩家的“频谱偏移疲劳”累积。这迫使玩家必须在不同相位区域间穿梭移动，不能固守一处，强化了最终战对玩家全机制调动能力的要求。
3.  **白噪音安全区**：场景中会随机、短暂地出现“白噪音区域”。这些区域在视觉上表现为纯粹的、混乱的雪花点。在这些区域内，所有相位规则暂时失效，玩家和敌人都会受到持续的微量伤害，但这里也是唯一能快速降低“频谱偏移疲劳”的地方。它既是避难所，也是危险地带，玩家需要在“躲避相位规则”和“承受持续伤害”之间做出抉择。 |
