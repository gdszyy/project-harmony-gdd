## 0. 设计总论：以"恰到好处的引导"取代"被动的发现"

### 0.1. 本文档的定位

本文档是对《关卡与Boss机制设计文档 v2.0》的全面重写。v2.0 侧重于宏观的"音乐史叙事"与"风格排斥"框架，但缺少**关卡内波次级别的精确设计**——即每一波敌人的构成如何精确引导玩家使用特定机制。本文档将在保留 v2.0 宏观框架的基础上，为每一章的每一个关卡波次提供**具体的敌人配置、预期玩家操作和机制教学目标**，并确保所有设计均精确对应 GDD 及数值设计文档 [1, 2, 3, 4, 5] 中定义的规则与数值，使关卡设计真正做到"恰到好处"地引导玩家循序渐进地掌握所有核心机制。

### 0.2. 核心设计哲学

本文档的设计遵循三条核心原则：

**原则一：环境即教程（Environment as Tutorial）。** 不通过文字弹窗教学，而是通过敌人构成、波次节奏和场景压力，让玩家在实战中"自然发现"最优操作。例如，当一波敌人恰好由 C、E、G 三种属性的目标组成时，玩家会自然地尝试同时攻击它们，从而"发现"大三和弦。

**原则二：先安全试错，再实战检验（Safe Sandbox → Combat Test）。** 每个新机制的引入都遵循"教学波→练习波→考试波"的三段式结构。教学波提供低压力环境让玩家尝试新操作；练习波增加复杂度但仍有容错；考试波则要求玩家熟练运用该机制才能通过。

**原则三：辩证否定链（Dialectical Negation Chain）。** 每一章的"最优策略"都会在下一章变成"被惩罚的行为"，迫使玩家不断突破舒适区。这条链条贯穿七章：单音→和弦→复调→简约→动态→即兴→频率。

| :--- | :--- | :--- | :--- |

### 0.4. 波次设计标准模板

为确保设计的一致性和可实现性，每个关卡的波次设计均遵循以下标准模板：

| 字段 | 说明 |
| :--- | :--- |
| **波次编号** | 该章节内的波次序号 |
| **波次类型** | 教学波 / 练习波 / 考试波 / Boss 前奏波 |
| **敌人配置** | 具体的敌人类型、数量和入场方式 |
| **场景条件** | 特殊的场景状态（如 BPM 变化、相位限制等） |

| **容错机制** | 如果玩家未能理解，系统提供的提示或降压措施 |

## 第一章：律动尊者·毕达哥拉斯 — 节奏与音符的启蒙

### 1.0. 章节概述

| 项目 | 内容 |
| :--- | :--- |
| **历史对标** | 古希腊·毕达哥拉斯学派 |
| **核心冲突** | 秩序 vs. 混沌 |
| **场景** | 由纯粹光线构成的圆形殿堂，地面为脉冲网格 |
| **教学目标** | 节奏型（均匀/附点/休止符）、白键音符四维差异、完美卡拍 |
| **出场敌人** | Static（底噪）、Pulse（脉冲） |
| **Boss** | 律动尊者·毕达哥拉斯 |
| **通关奖励** | 解锁和弦炼成系统、大三和弦配方 |

### 1.1. 关卡波次设计

#### 波次 1-1：初识节拍（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 4 只 Static，从场景四个方向缓慢进入，间隔 2 拍生成一只 |
| **场景条件** | BPM=100（较慢），地面脉冲网格清晰可见，每拍闪烁一次 |
| **设计意图** | 让玩家感受"节拍"的存在。Static 的移动与网格脉冲同步，玩家自然地在脉冲闪烁时按下攻击键 |
| **预期玩家操作** | 玩家使用初始装备的 C 音符（均衡型，DMG 30），在节拍上发射弹体击杀 Static |
| **容错机制** | Static 移动极慢（速度 80），即使玩家节奏不准也能在被接触前击杀。屏幕底部显示节拍指示条，完美卡拍时产生金色闪光反馈 |

#### 波次 1-2：音符差异（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 6 只 Static，分两组：3 只从远处（800px）缓慢接近，3 只从近处（300px）快速接近 |
| **场景条件** | 玩家通过升级获得了 D 音符（极速远程，SPD 1000px/s）和 G 音符（爆发伤害，DMG 50） |
| **设计意图** | 远处的 Static 移动缓慢但距离远，适合用 D 音符（高速度、长射程 1500px）狙击；近处的 Static 速度快但血量低，适合用 G 音符（高伤害 50、短射程 600px）近距离爆发。让玩家体会不同音符的四维差异带来的战术选择 |
| **预期玩家操作** | 玩家在序列器中交替编排 D 和 G 音符：用 D 处理远程威胁，用 G 处理近身威胁。或者手动施法槽 1 放 D、槽 2 放 G，根据情况切换 |
| **容错机制** | 即使全用 C 音符（DMG 30, SPD 600px/s）也能通过，但会明显感到"远处的打不到、近处的来不及"的不适感，暗示需要使用不同音符 |

#### 波次 1-3：完美卡拍（练习波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 8 只 Static 蜂群（群体加速生效），从同一方向涌入 |
| **场景条件** | BPM 提升至 110。完美卡拍时伤害 ×1.5、击退 ×2.5 [5] |
| **设计意图** | 蜂群数量多且会群体加速（最高 1.6x），单纯的普通攻击难以阻止推进。完美卡拍的 1.5 倍伤害和 2.5 倍击退是控制蜂群的关键。让玩家主动追求完美卡拍的精准度 |
| **预期玩家操作** | 玩家紧盯节拍指示条，在完美时刻释放攻击，利用高额击退将蜂群推回，争取更多输出时间 |
| **容错机制** | 蜂群 HP 较低（每只 30），即使不完美卡拍，用 G 音符（50 伤害）也能一击击杀。但如果不利用击退控制距离，后续的 Static 会越来越快地逼近 |

#### 波次 1-4：节奏型入门——休止符的力量（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 2 只 Wall（音墙），HP 200 + 50 护盾，从正前方缓慢推进 |
| **场景条件** | 序列器提示玩家可以在某些拍位编入"休止符"。UI 高亮显示休止符图标 |
| **设计意图** | Wall 的 HP 极高（200+50 护盾），用普通 C 音符（30 伤害）需要至少 9 发才能击杀。但如果在 4 拍序列中编入 2 个休止符，剩余 2 个音符各获得 +1.0 伤害和 +1.0 大小的加成（每个休止符 +0.5）[1]，使得 G 音符的伤害从 50 提升至 60，大小从 16px 提升至 24px，大幅提高效率。这是玩家第一次体会"少即是多"——留白反而增强输出 |
| **预期玩家操作** | 玩家在序列器中编排"G - 休止 - G - 休止"的节奏型，用增强后的 G 音符高效击破 Wall 的护盾和本体 |
| **容错机制** | Wall 移动极慢（速度 25），给玩家充足的时间尝试不同的序列器编排。如果玩家 15 秒内未编入休止符，屏幕边缘会出现柔和的提示光效，指向休止符按钮 |

#### 波次 1-5：节奏型进阶——附点节奏（练习波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只 Wall + 6 只 Static 护卫（围绕 Wall 移动） |
| **场景条件** | 通过升级解锁"附点节奏"节奏型。附点节奏效果：飞行速度-1，但伤害+1，并获得击退 [1] |
| **设计意图** | Wall 被 Static 护卫包围，普通弹体难以穿过 Static 群击中 Wall。附点节奏的"击退"效果可以将 Static 推开，为后续弹体打开通路。同时附点节奏的伤害加成有助于更快击破 Wall 护盾 |
| **预期玩家操作** | 玩家将序列器切换为附点节奏，利用击退效果清理 Static 护卫，然后集中火力击破 Wall |
| **容错机制** | Static 护卫的 HP 仅 30，即使不用附点节奏，也可以先用普通攻击清理护卫再打 Wall，只是效率较低 |

#### 波次 1-6：综合运用（考试波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 混合波：10 只 Static 蜂群 + 2 只 Pulse + 1 只 Wall，分三个方向入场 |
| **场景条件** | BPM=120（标准速度）。Pulse 每 4 拍释放 8 发环形弹幕 [5] |
| **设计意图** | 综合考验玩家对音符选择、节奏型和完美卡拍的掌握。Static 蜂群需要范围控制（E 音符，SIZE 32px）或完美卡拍击退；Pulse 的弹幕需要走位躲避，其 4 拍蓄力周期暗示玩家利用间隙输出；Wall 需要休止符增强的高伤害音符击破 |
| **预期玩家操作** | 玩家根据威胁优先级动态调整：手动施法槽 1 用 E（大范围）处理蜂群，槽 2 用 G（高伤害）处理 Wall，序列器编排带休止符的附点节奏。在 Pulse 蓄力间隙集中输出 |
| **容错机制** | 三个方向的敌人不会同时到达，给玩家逐一处理的时间窗口 |

### 1.2. Boss 战：律动尊者·毕达哥拉斯

#### Boss 概述

| 项目 | 内容 |
| :--- | :--- |
| **形态** | 位于场景中心的多层旋转光环几何体，不移动 |
| **HP** | 阶段一 800 / 阶段二 1200 |
| **核心机制** | 克拉尼图形致死区域 + 节点安全区 |
| **时代特征** | 【绝对频率】——地面生成不断变化的克拉尼图形，线条致死，节点安全 |
| **风格排斥** | 惩罚无效输入（胡乱按键产生"噪音"，Boss 全屏微弱伤害 + 屏幕浮现"亵渎！"） |

#### 阶段一：简单图形（HP 800→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **八度共振** | 地面生成最简单的克拉尼图形（同心圆），安全节点为 4 个固定点 | 站在节点上输出，节点之间的移动需要卡准图形切换的间隙 |
| **五度震荡** | 图形变为更复杂的花瓣形，安全节点增至 6 个但位置变化 | 观察图形变化规律（每 8 拍切换一次），提前移动到新节点 |
| **节拍考验** | Boss 每 4 拍释放一次全屏脉冲，只有在完美卡拍时刻攻击 Boss 才能造成伤害（其他时刻攻击被"和谐护盾"反弹） | 严格在完美卡拍时刻释放攻击，利用 G 音符的高伤害（50）最大化每次攻击的收益 |

#### 阶段二：复杂图形（HP 1200→0，Boss 进入狂暴）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **四度叠加** | 两组克拉尼图形叠加，安全区域大幅缩小 | 需要精准走位，利用 D 音符的远射程（1500px）在安全区外输出 |
| **不和谐脉冲** | Boss 释放不规则节奏的脉冲，打乱玩家的节拍感 | 玩家需要适应变化的节奏，而非机械地按固定间隔攻击 |
| **终焉和弦** | Boss HP 低于 20% 时，所有图形同时激活，仅留 2 个安全节点，且每 4 拍切换位置 | 这是对第一章所有技能的终极考验：精准走位 + 完美卡拍 + 正确音符选择 |

#### Boss 击败奖励

> **解锁：和弦炼成系统。** 击败毕达哥拉斯后，玩家获得"和弦炼成台"，并获得足够的 C、E、G 音符来合成第一个**大三和弦**（C-E-G → 强化弹体）。系统会引导玩家进行第一次和弦炼成，为第二章做准备。

## 第二章：圣咏宗师·圭多 — 和弦的觉醒

### 2.0. 章节概述

| 项目 | 内容 |
| :--- | :--- |
| **历史对标** | 中世纪·圭多·达雷佐 |
| **核心冲突** | 单声 vs. 复调萌芽 |
| **场景** | 高耸幽暗的哥特式大教堂，强烈混响，地面刻印五线谱 |
| **教学目标** | 大三/小三和弦、和弦炼成、听感疲劳（音高/转移/模式疲劳） |
| **出场敌人** | Static、Silence（首次登场）、唱诗班（Static 编队变体） |
| **Boss** | 圣咏宗师·圭多 |
| **通关奖励** | 解锁增三和弦、减三和弦配方 |

### 2.1. 关卡波次设计

#### 波次 2-1：单音的局限（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 个唱诗班编队（5 只 Static 保持固定间距同步移动，如同一体） |
| **场景条件** | 唱诗班编队的 5 只 Static 紧密排列，总 HP 为 150（5×30），但它们共享一个"编队护盾"（额外 100 HP），只有**范围攻击或同时命中多个成员**才能有效削减护盾 |
| **设计意图** | 单音弹体一次只能命中一个 Static，对编队护盾的削减效率极低。玩家会感到"单音打不动这群敌人"的挫败感，自然想到使用刚解锁的和弦 |
| **预期玩家操作** | 玩家使用在第一章结尾合成的**大三和弦**（强化弹体）[1]，其增大的弹体可以同时命中多个编队成员，高效削减编队护盾 |
| **容错机制** | 唱诗班移动极慢（速度 30），给玩家充足时间尝试。如果玩家持续使用单音攻击 20 秒未使用和弦，UI 会柔和地高亮手动施法槽中的和弦法术 |

#### 波次 2-2：和弦炼成实战（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 暂无敌人。系统触发一次强制升级，奖励玩家 A、C、E 三个音符 |
| **场景条件** | 炼成台界面自动弹出，预览区显示"A-C-E → 小三和弦 → DOT 弹体（持续伤害）" [1] |
| **设计意图** | 引导玩家进行第二次和弦炼成，让玩家理解不同音符组合产生不同和弦类型。小三和弦的 DOT（持续伤害）效果与大三和弦的强化弹体形成对比，让玩家体会和弦类型的多样性 |
| **预期玩家操作** | 玩家合成小三和弦，将其放入手动施法槽或序列器 |
| **容错机制** | 炼成台界面提供清晰的引导箭头和文字说明 |

#### 波次 2-3：大三 vs 小三的战术选择（练习波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 2 个唱诗班编队（从左右两侧进入）+ 1 只 Silence（从正前方缓慢接近） |
| **场景条件** | Silence 的静音光环（120px 半径）会使光环内法术伤害降低 40%，且每秒增加 0.08 疲劳度 [5] |
| **设计意图** | 唱诗班编队适合用大三和弦（强化弹体，高瞬间伤害）快速清理。Silence 则适合用小三和弦（DOT 弹体，持续伤害），因为 DOT 效果可以在玩家离开光环后继续造成伤害，避免长时间暴露在静音光环中。让玩家体会两种和弦的不同战术定位 |
| **预期玩家操作** | 先用大三和弦清理两侧唱诗班，再用小三和弦标记 Silence 后拉开距离，让 DOT 持续消耗 |
| **容错机制** | Silence 移动极慢（速度 35），即使玩家操作不当也有充足时间调整。击杀 Silence 会降低玩家 0.1 疲劳度，作为正向反馈 [5] |

#### 波次 2-4：听感疲劳初体验（教学波）

| :--- | :--- |
| **敌人配置** | 3 对小步舞曲舞者 + 持续生成的 Screech（尖啸） |
| **场景条件** | 通过升级获得大七和弦（护盾/治疗法阵）[1]。Screech 会在靠近 200px 时以 3x 速度冲刺，死亡时 80px 范围内造成 15 伤害 + 0.12 不和谐度 [5] |
| **设计意图** | Screech 的冲刺和死亡爆发使玩家面临持续的伤害压力。大七和弦的"护盾/治疗法阵"可以在玩家脚下创造一个持续回血的安全区域，抵消 Screech 的爆发伤害。让玩家理解大七和弦的核心价值——**防御与恢复** |
| **预期玩家操作** | 在自己的站位处放置大七和弦治疗法阵，然后用属七和弦法阵控制舞者路线，用单音或和弦处理冲刺的 Screech |
| **容错机制** | 治疗法阵的回血速度（2/秒）足以抵消 Screech 的零星爆发伤害 |

#### 波次 4-3：小七和弦——召唤构造（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只精英 Wall（HP 400 + 100 护盾）+ 持续从 Wall 身后涌出的 Static 蜂群 |
| **场景条件** | 通过升级获得小七和弦（召唤/构造）[1]。小七和弦可以召唤一个"节拍哨塔"——一个固定位置的自动攻击构造物 |
| **设计意图** | 精英 Wall 需要持续输出才能击破，但 Wall 身后的 Static 蜂群会不断分散玩家注意力。小七和弦召唤的"节拍哨塔"可以自动攻击 Static，让玩家专注于击破 Wall。这是玩家第一次体验"多声部协作"——自己是主旋律，召唤物是伴奏 |
| **预期玩家操作** | 在 Static 涌出的路径上放置小七和弦召唤物（节拍哨塔），让它自动处理 Static。玩家自己集中火力用 D→T 和弦进行的爆发效果击破 Wall |
| **容错机制** | 节拍哨塔的攻击力足以在 Static 接近前将其击杀 |

#### 波次 4-4：和弦进行完整度奖励（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只超级精英 Wall（HP 800 + 200 护盾），移动速度正常 |
| **场景条件** | UI 首次显示"和弦进行完整度"指示器。连续释放 2 个和弦的进行效果为 ×1.0 倍率，3 个和弦为 ×1.5，4 个和弦为 ×2.0 [1] |
| **设计意图** | 超级精英 Wall 的 HP 极高（800+200），即使用 D→T 爆发也需要多次才能击杀。但如果玩家能连续释放 4 个和弦构成完整进行（如：大三T → 增三D → 减三D → 大三T），最后的爆发效果将获得 ×2.0 倍率，大幅提高效率。让玩家理解**和弦进行的长度越长，奖励越大** |
| **预期玩家操作** | 玩家规划一个 4 和弦进行序列：例如"大三（T）→ 减三（D）→ 属七（D）→ 大三（T）"，触发 T→D 蓄力 + D→D 加速 + D→T 爆发（×2.0），对 Wall 造成毁灭性伤害 |
| **容错机制** | 即使只完成 2 和弦进行（×1.0），多次重复也能击杀 Wall，只是需要更长时间 |

#### 波次 4-5：密度疲劳与留白——少即是多（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 持续涌入的 Static 蜂群（每秒 3 只，持续 40 秒） |
| **场景条件** | 疲劳系统的**密度疲劳 (F_density)** 和 **留白疲劳 (F_rest)** 首次被激活并在 UI 上高亮显示 [3]。如果玩家连续施法不停歇，这两个维度会快速上升，导致所有法术效果大幅削弱 |
| **设计意图** | 持续 40 秒的蜂群涌入会诱使玩家不停施法。但如果不留出"呼吸空间"，密度疲劳和留白缺失疲劳会在约 20 秒后将 AFI 推至 MODERATE（伤害 ×0.65），反而降低了清理效率。玩家需要学会"有节奏地施法"——每 3-4 次攻击后留出 1-2 拍的休息，让疲劳自然衰减。这与第一章学到的"休止符增强"形成呼应——**留白不仅增强单次伤害，还能维持长期输出** |
| **预期玩家操作** | 在序列器中编排"攻击-攻击-攻击-休止"的节奏模式，或在手动施法时有意识地在每 3-4 次攻击后暂停 1-2 拍。保持密度疲劳和留白疲劳在低水平 |
| **容错机制** | Static 的 HP 低（30），即使伤害被削弱也能击杀。UI 会在密度疲劳上升时显示明确的警告："放慢施法节奏，给音乐留出空间" [3] |

#### 波次 4-6：结构之美（考试波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 4 对小步舞曲舞者 + 2 只精英 Wall + 持续的 Screech 和 Static |
| **场景条件** | BPM=120。所有机制完全激活。场景中的镜子会反射 Boss 的攻击预告（为 Boss 战做铺垫） |
| **设计意图** | 综合考验七和弦运用、和弦进行完整度、密度疲劳管理。舞者需要属七法阵控制，Screech 需要大七治疗法阵抵消爆发伤害，Wall 需要完整度 ×2.0 的和弦进行爆发击破，全程需要控制施法密度避免疲劳 |
| **预期玩家操作** | 放置属七法阵控制舞者路线 → 放置大七治疗法阵保护自身 → 放置小七召唤物处理 Static → 规划 4 和弦进行击破 Wall → 全程保持施法节奏的"呼吸感" |
| **容错机制** | 敌人分批入场，给玩家逐步建立防御体系的时间 |

### 4.2. Boss 战：古典完形·莫扎特

#### Boss 概述

| 项目 | 内容 |
| :--- | :--- |
| **形态** | 穿着华丽燕尾服的贵公子，手持水晶指挥棒 |
| **HP** | 呈示部 1000 / 发展部 1500 / 再现部 2000 |
| **核心机制** | 奏鸣曲式三段攻击 + 繁复的诅咒 |
| **时代特征** | 【奏鸣曲式力场】——战斗分为呈示部、发展部、再现部三段 |
| **风格排斥** | 【繁复的诅咒】——同时存在 2 个以上召唤物，或使用密度极高的节奏型时，Boss 清除所有召唤物并增加疲劳度 |

#### 呈示部（HP 1000→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **主题 A：冲刺斩击** | Boss 进行 3 次快速直线冲刺斩击，每次之间间隔 2 拍 | 利用 2 拍间隙横向闪避，在第 3 次斩击后的恢复期输出 |
| **主题 B：圆舞曲弹幕** | Boss 旋转释放螺旋形弹幕，弹幕沿 3/4 拍节奏扩散 | 跟随 3/4 拍节奏在弹幕间隙穿行 |
| **繁复检测** | Boss 检测玩家的召唤物数量和施法密度 | 本阶段最多保留 1 个召唤物，施法节奏保持"呼吸感" |

#### 发展部（HP 1500→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **主题变形** | Boss 将主题 A 和 B 组合：在释放螺旋弹幕的同时进行冲刺斩击 | 需要同时处理两种威胁，走位难度大幅提升 |
| **镜像攻击** | 场景中的镜子会反射 Boss 的攻击，形成对称的双重威胁 | 利用对称性预判镜像攻击的轨迹 |
| **脆弱窗口** | Boss 每完成一组变形攻击后，会优雅地鞠躬 2 秒（脆弱期） | 在鞠躬期间释放完整的 4 和弦进行（×2.0 倍率）最大化伤害 |

#### 再现部（HP 2000→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **华彩乐章** | 主题 A 和 B 的速度、范围、伤害全面提升 50% | 需要更精准的走位和更高效的输出 |
| **完美终止** | Boss HP 低于 10% 时，释放"完美终止式"——全屏缓慢收缩的光环，只有在光环完全收缩前击杀 Boss 才能避免全灭 | 需要在压力下快速释放最高效的和弦进行组合 |

#### Boss 击败奖励

> **解锁：减七和弦（天降打击，极度不稳定→毁灭打击，不和谐度 6.50）和切分节奏（发射时玩家向后微小位移）。** 减七和弦的高不和谐度将在第五章成为核心教学内容，切分节奏则为第五章的动态 BPM 适应做准备。

## 第五章：狂想者·贝多芬 — 不和谐与动态

### 5.0. 章节概述

| 项目 | 内容 |
| :--- | :--- |
| **历史对标** | 浪漫主义·路德维希·凡·贝多芬 |
| **核心冲突** | 形式 vs. 情感 |
| **场景** | 风暴肆虐的荒野，崩坏的废墟，动态天气（雨、闪电） |

### 5.1. 关卡波次设计

#### 波次 5-1：减七和弦——天降打击（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只精英 Pulse（HP 120），被 4 只 Wall 围困在中央，Wall 之间留有狭窄缝隙 |
| **场景条件** | 玩家刚获得减七和弦（天降打击，延迟 2 拍，伤害倍率 3.0x，不和谐度 6.50）[2]。精英 Pulse 会在 Wall 之间快速移动，难以直接瞄准 |
| **设计意图** | 减七和弦的"天降打击"是第一个拥有延迟和范围标记的法术。玩家需要预判精英 Pulse 的移动位置，提前放置打击标记。这让玩家首次体验到**预判性攻击**和**高风险高回报**——2 拍的延迟是风险，3.0x 的伤害倍率是回报 |
| **预期玩家操作** | 观察精英 Pulse 的移动规律，在其路径上提前 2 拍释放减七和弦，利用其高额伤害一次性击杀 |
| **容错机制** | Wall 之间的缝隙很窄，精英 Pulse 的移动范围有限，降低了预判难度。即使未命中，也可以用其他穿透性法术（如 C# 修饰的弹体）穿过缝隙攻击 |

#### 波次 5-2：不和谐度初体验（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只超级精英 Wall（HP 800 + 200 护盾） |
| **场景条件** | 不和谐度 UI 首次完整显示。每次使用减七和弦都会增加 6.50 不和谐度。当不和谐度超过 5.0 时，会触发"轻度失谐"（屏幕边缘轻微扭曲，所有治疗效果降低 20%）[1] |
| **设计意图** | 减七和弦的强大威力伴随着高昂的不和谐度代价。如果玩家连续使用减七和弦攻击超级精英 Wall，会在第二次使用后触发"轻度失谐"。这让玩家理解**不和谐度是一种需要主动管理的负面资源**。不和谐度会随时间自然衰减（-0.5/秒）[2]，或通过使用协和和弦（如大三和弦）快速降低 |
| **预期玩家操作** | 玩家在使用一次减七和弦后，穿插使用 2-3 次大三和弦或大七和弦来主动降低不和谐度，然后再使用下一次减七和弦，形成"紧张→解决→再紧张"的循环 |
| **容错机制** | "轻度失谐"的惩罚较弱，主要起警示作用。UI 会在不和谐度条旁边显示"使用协和和弦可快速降低"的提示 |

#### 波次 5-3：动态 BPM——弹性速度（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 风暴元素首次登场：2 只。它们会周期性地释放"风眼"，改变区域内的 BPM |
| **场景条件** | 场景初始 BPM=120。当玩家进入风眼区域时，BPM 会在 100-140 之间动态变化（`ritardando` 减速 和 `accelerando` 加速） |
| **设计意图** | 这是玩家第一次面对非固定的 BPM。玩家需要根据变化的节拍指示器调整自己的攻击节奏，而非依赖肌肉记忆。这为 Boss 战的"弹性速度"机制做准备 |
| **预期玩家操作** | 玩家紧盯节拍指示器，根据其闪烁快慢调整攻击频率，在动态节奏中保持完美卡拍 |
| **容错机制** | 风眼的范围固定且有清晰的视觉边界，玩家可以选择不进入风眼，在 120 BPM 的稳定区域与敌人周旋，但风暴元素会持续释放闪电链攻击外部区域，鼓励玩家进入风眼挑战 |

#### 波次 5-4：切分节奏——反拍攻击（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 3 只 Screech + 持续生成的 Static |
| **场景条件** | 通过升级获得"切分节奏"。切分节奏效果：在弱拍（off-beat）位置攻击，发射时玩家会向后微小位移，且该次攻击必定暴击 [1] |
| **设计意图** | Screech 的冲刺攻击极快，难以正面应对。切分节奏的"反拍后撤"提供了一种全新的"攻击即闪避"的战术。玩家可以在 Screech 冲刺的瞬间，利用切分节奏攻击，后撤躲避冲刺的同时对其造成暴击伤害。让玩家理解**节奏不仅可以用来攻击，还可以用来防御和位移** |
| **预期玩家操作** | 玩家将序列器切换为切分节奏，在 Screech 冲刺时背对冲刺方向释放攻击，利用后撤位移躲避伤害并反击 |
| **容错机制** | 后撤位移的距离足以躲开 Screech 的冲刺路径。即使操作失败，Screech 的单次伤害也不致命 |

#### 波次 5-5：命运的敲门（考试波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 2 只风暴元素 + 4 只 Screech + 1 只精英 Pulse |
| **场景条件** | 动态 BPM + 不和谐度系统完全激活 |
| **设计意图** | 综合考验。风暴元素制造动态 BPM 环境；Screech 需要切分节奏处理；精英 Pulse 需要减七和弦预判打击；全程需要管理不和谐度。这是对玩家动态适应能力的终极考验 |
| **预期玩家操作** | 进入风眼适应动态 BPM → 用切分节奏处理 Screech → 预判 Pulse 位置释放减七和弦 → 穿插大七和弦治疗并降低不和谐度 → 形成一套完整的动态战斗循环 |
| **容错机制** | 敌人分批入场 |

### 5.2. Boss 战：狂想者·贝多芬

#### Boss 概述

| 项目 | 内容 |
| :--- | :--- |
| **形态** | 头发狂乱、衣衫不整的巨人，被锁链束缚在废墟王座上 |
| **HP** | 阶段一 2500 / 阶段二 3000 |
| **核心机制** | 命运动机（短-短-短-长节奏攻击）+ 弹性速度 |
| **时代特征** | 【弹性速度】——Boss 周期性地进入 `Accelerando` (加速) 和 `Ritardando` (减速) 状态，改变全局 BPM |
| **风格排斥** | 【英雄的蔑视】——玩家在 `Accelerando` 阶段使用低不和谐度和弦（如大三），或在 `Ritardando` 阶段使用高不和谐度和弦（如减七），会激怒 Boss，使其下一次攻击伤害翻倍 |

#### 阶段一：月光（HP 2500→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **命运动机** | Boss 挥舞锁链，以"短-短-短-长"的节奏进行四连击，最后一下为大范围冲击波 | 根据节奏提示音和视觉预警进行四次闪避，在第四击后的硬直期输出 |
| **Ritardando** | Boss 进入减速状态，全局 BPM 降至 80。此时是使用高风险法术的安全期 | 在减速阶段释放减七和弦（延迟 2 拍，此时时间更长，更容易命中），并规划完整和弦进行 |
| **Accelerando** | Boss 进入加速状态，全局 BPM 升至 160。此时需要快速反应 | 在加速阶段使用低延迟的单音或简单和弦进行快速攻击，避免使用减七和弦等高延迟法术 |

#### 阶段二：暴风雨（HP 3000→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **命运风暴** | 命运动机的攻击变为全屏随机落雷，节奏不变 | 需要在落雷的缝隙中穿行，同时保持对"短-短-短-长"节奏的判断 |
| **不和谐共鸣** | 玩家的不和谐度越高，落雷的密度和伤害越高 | 必须积极管理不和谐度，在减速阶段用协和和弦（大七/大三）清空不和谐度条 |
| **英雄的挑战** | Boss HP 低于 20% 时，进入永久的 `Accelerando` 状态（BPM=180），并持续召唤风暴元素 | 这是对玩家反应速度和资源管理能力的极限考验。利用切分节奏的后撤位移躲避攻击，在极限速度下完成输出 |

#### Boss 击败奖励

> **解锁：摇摆节奏（Swing Rhythm）和扩展和弦（九和弦、十一和弦）配方。** 摇摆节奏将为第六章的爵士乐主题提供核心律动，而扩展和弦的引入（5 音和弦）将法术的复杂度和威力提升到全新高度，并与频谱相位系统产生联动。

## 第六章：摇摆公爵·艾灵顿 — 摇摆与即兴

### 6.0. 章节概述

| 项目 | 内容 |
| :--- | :--- |
| **历史对标** | 爵士乐·艾灵顿公爵 |
| **核心冲突** | 谱写 vs. 即兴 |
| **场景** | 烟雾缭绕、霓虹闪烁的 1930 年代爵士俱乐部 |
| **教学目标** | 摇摆节奏、扩展和弦（九/十一和弦）、频谱相位系统（入门） |
| **出场敌人** | 即兴独奏者（Improvising Soloist，新敌人）、保镖（Bouncer，音墙变体） |
| **Boss** | 摇摆公爵·艾灵顿 |
| **通关奖励** | 频谱相位系统完全解锁、音色相位增益 |

### 6.1. 关卡波次设计

#### 波次 6-1：摇摆节奏（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 即兴独奏者首次登场：1 只。它会演奏一段快速的、不可预测的旋律线（表现为高速弹幕），但其移动遵循摇摆节奏（一长一短） |
| **场景条件** | 玩家刚获得"摇摆节奏"。切换后，节拍指示器变为"长-短-长-短"的摇摆律动 |
| **设计意图** | 即兴独奏者的弹幕难以躲避，但其本体的移动遵循可预测的摇摆节奏。玩家需要切换到摇摆节奏，让自己的攻击与敌人的移动同步，才能有效命中。让玩家感受**摇摆节奏的独特律动** |
| **预期玩家操作** | 玩家将序列器切换为摇摆节奏，跟随新的律动攻击即兴独奏者 |
| **容错机制** | 即兴独奏者的弹幕伤害较低，且有明显的音高变化作为预警 |

#### 波次 6-2：扩展和弦——属九和弦（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 3 只即兴独奏者，在场景中高速移动并演奏复杂的旋律线 |
| **场景条件** | 通过升级获得属九和弦（5 音和弦，风暴区域，区域内敌人减速 30%，不和谐度 5.00）[2] |
| **设计意图** | 即兴独奏者的高速移动使其难以被持续锁定。属九和弦的"风暴区域"可以创造一个大范围的减速场，将其困在其中，大幅降低处理难度。让玩家体验**扩展和弦强大的控场能力** |
| **预期玩家操作** | 在即兴独奏者移动路径的交点释放属九和弦，创造减速区域，然后在区域内用其他法术轻松消灭它们 |
| **容错机制** | 减速区域持续时间长（8 秒），即使第一次释放位置不佳，也有机会在第二次尝试时成功 |

#### 波次 6-3：频谱相位系统入门（教学波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只保镖（Bouncer，音墙变体，免疫所有物理伤害） |
| **场景条件** | 频谱相位系统首次解锁（仅高通/低通切换，无疲劳惩罚）。保镖在**全频相位**下物理免疫，但在**高通相位**下，其护盾消失，本体暴露 [4] |
| **设计意图** | 这是玩家第一次体验相位切换。面对物理免疫的保镖，任何在全频相位下的攻击都无效。UI 会提示玩家尝试新的按键（相位切换键）。当玩家切换到高通相位时，会发现保镖的形态发生了变化，变得可以被伤害了。让玩家理解**相位切换是解决特定敌人的钥匙** |
| **预期玩家操作** | 玩家攻击保镖发现无效 → 根据 UI 提示切换到高通相位 → 发现保镖变得脆弱并将其击杀 |
| **容错机制** | 保镖移动极慢，且只会推开玩家而不造成伤害，给玩家充足的时间安全地尝试新机制 |

#### 波次 6-4：高通 vs 低通（练习波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 1 只即兴独奏者（高频威胁）+ 1 只 Pulse（低频威胁） |
| **场景条件** | 即兴独奏者的弹幕在**低通相位**下会消失；Pulse 的环形弹幕在**高通相位**下会消失 [4] |
| **设计意图** | 玩家同时面对两种不同频率的威胁。如果停留在全频相位，会被两种弹幕同时攻击。玩家需要学会根据威胁类型动态切换相位：面对即兴独奏者时切换到低通相位（使其弹幕无效），面对 Pulse 时切换到高通相位（使其弹幕无效）。让玩家理解**相位切换的核心战术价值——隔离威胁** |
| **预期玩家操作** | 面对即兴独奏者时切换到低通相位进行攻击 → Pulse 准备释放弹幕时切换到高通相位躲避并反击 → 在两个相位间来回切换，将战场分割处理 |
| **容错机制** | 两种敌人的攻击都有明显预警，给玩家充足的切换时间 |

#### 波次 6-5：大乐队（考试波）

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 2 只即兴独奏者 + 2 只保镖 + 持续生成的 Pulse |
| **场景条件** | 摇摆节奏 + 扩展和弦 + 相位切换全部激活 |
| **设计意图** | 综合考验。保镖需要高通相位处理；即兴独奏者和 Pulse 需要在不同相位间切换隔离威胁；多个高速移动的目标需要属九和弦的减速场控制；全程需要在摇摆节奏下进行 |
| **预期玩家操作** | 切换到摇摆节奏 → 释放属九和弦减速所有敌人 → 切换到高通相位击杀保镖 → 在高通/低通相位间切换处理独奏者和 Pulse |
| **容错机制** | 敌人分批入场 |

### 6.2. Boss 战：摇摆公爵·艾灵顿

#### Boss 概述

| 项目 | 内容 |
| :--- | :--- |
| **形态** | 优雅地坐在钢琴前的乐队指挥，身后是他的大乐队（由幽灵乐手组成） |
| **HP** | 阶段一 3000 / 阶段二 4000 |
| **核心机制** | 声部轮奏（Call and Response）+ 即兴独奏 |
| **时代特征** | 【Call and Response】——Boss 指挥不同声部（铜管/木管/打击乐）轮流攻击，玩家需要攻击正在演奏的声部才能对 Boss 造成伤害 |
| **风格排斥** | 【摇摆的灵魂】——玩家未使用摇摆节奏攻击时，伤害降低 70% |

#### 阶段一：乐队合奏（HP 3000→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **铜管组（高频）** | 铜管组乐手起立演奏，释放密集的穿透性高频弹幕 | 切换到**低通相位**使弹幕无效，同时攻击铜管组乐手 |
| **木管组（中频）** | 木管组乐手起立演奏，释放追踪型中频弹幕 | 在**全频相位**下走位躲避，同时攻击木管组乐手 |
| **打击乐组（低频）** | 打击乐手起立演奏，释放全屏的低频震荡波 | 切换到**高通相位**使震荡波无效，同时攻击打击乐手 |

#### 阶段二：即兴独奏（HP 4000→0）

| 攻击模式 | 描述 | 玩家应对 |
| :--- | :--- | :--- |
| **萨克斯独奏** | Boss 邀请萨克斯手进行即兴独奏，释放极其复杂的、混合了三个频段的弹幕 | 这是对相位切换速度的极限考验。玩家需要在三个相位间快速切换来躲避不同频段的弹幕 |
| **钢琴独奏** | Boss 亲自进行钢琴独奏，双手分别演奏不同声部，对应不同相位的攻击 | 需要同时观察 Boss 的双手动作，预判并切换到正确的相位 |
| **终焉合奏** | Boss HP 低于 20% 时，所有声部同时起立演奏，形成三层相位叠加的弹幕风暴 | 此时必须使用**减十三和弦（终焉乐章）**[2]——延迟 4 拍后全屏毁灭打击，其高额伤害和全屏范围是终结战斗的唯一途径 |

#### Boss 击败奖励

> **解锁：频谱相位系统完全功能**（包括频谱偏移疲劳和相位能量系统），**音色相位增益**（不同音色在特定相位获得加成）。这标志着玩家已完全掌握所有核心机制，准备迎接最终的挑战。

## 第七章：合成主脑·噪音 — 频率与终焉

### 7.0. 章节概述

| 项目 | 内容 |
| :--- | :--- |
| **历史对标** | 电子音乐、噪音音乐、故障艺术 |
| **核心冲突** | 音乐 vs. 噪音 |
| **场景** | 抽象的数字空间，由数据流、故障像素和示波器波形构成 |
| **教学目标** | 频谱相位系统（精通）、音色相位增益、频谱偏移疲劳管理 |
| **出场敌人** | 所有先前敌人的相位扭曲变体 |
| **Boss** | 合成主脑·噪音 |
| **通关奖励** | 游戏通关，解锁“自由演奏”模式 |

### 7.1. 关卡波次设计

本章节不再有传统的教学波，所有波次均为对玩家全机制掌握程度的综合考验。

#### 波次 7-1：相位失调

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 2 只 Silence（低通相位极度危险）+ 4 只 Screech（高通相位极度危险） |
| **场景条件** | 频谱偏移疲劳系统完全激活。长时间停留在单一极端相位（高通/低通）会导致频谱偏移疲劳快速累积，超过 0.7 时触发“频谱失调”全局 Debuff（屏幕扭曲，音色失效）[3] |
| **设计意图** | Silence 在低通相位下光环范围和疲劳效果加倍，Screech 在高通相位下变为瞬移电弧极难处理。这迫使玩家不能长时间停留在任何一个极端相位，必须频繁返回**全频相位**来降低频谱偏移疲劳。让玩家理解**全频相位是安全港和重置点** |
| **预期玩家操作** | 在高通相位处理 Silence 核心 → 返回全频相位休整 → 在低通相位压制 Screech → 返回全频相位休整。形成"全频 → 爆发 → 全频"的战术循环 |

#### 波次 7-2：音色增益

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 4 只即兴独奏者（高机动性）+ 2 只保镖（高护甲） |
| **场景条件** | 音色相位增益激活。例如，弹拨系音色在高通相位下伤害+50%，打击系音色在低通相位下击退/眩晕效果翻倍 [4] |
| **设计意图** | 鼓励玩家根据战术需求选择合适的音色和相位组合。面对高机动性的独奏者，可以在高通相位下使用弹拨系音色（如吉他、竖琴）获得伤害加成，快速秒杀。面对高护甲的保镖，可以在低通相位下使用打击系音色（如钢琴、钟）的强化击退效果将其推入陷阱或控制住 |
| **预期玩家操作** | 装备弹拨系音色，切换到高通相位处理独奏者；装备打击系音色，切换到低通相位控制保镖 |

#### 波次 7-3：最终的交响

| 字段 | 内容 |
| :--- | :--- |
| **敌人配置** | 所有先前 Boss 的"回响"（弱化版），按顺序出场：毕达哥拉斯的克拉尼图形，圭多的圣咏音墙，巴赫的赋格迷宫，莫扎特的奏鸣曲式攻击，贝多芬的命运动机 |
| **场景条件** | 所有机制完全激活 |
| **设计意图** | 这是 Boss 战前的最后考验，也是对玩家整个游戏学习过程的回顾。玩家需要回忆并运用之前学会的所有技巧来应对这些熟悉的攻击模式 |
| **预期玩家操作** | 综合运用节奏、和弦、和弦进行、不和谐度管理、相位切换等所有机制，应对所有 Boss 的经典招式 |

### 7.2. Boss 战：合成主脑·噪音

#### Boss 概述

| 项目 | 内容 |
| :--- | :--- |
| **形态** | 一个巨大的、不断变化的故障艺术几何体，核心是一个跳动的示波器 |
| **HP** | 10000 |
| **核心机制** | 波形战争（不同波形对应不同相位攻击）+ 频谱崩溃 |
| **时代特征** | 【波形战争】——Boss 在正弦波、方波、锯齿波、白噪音四种形态间切换，每种形态的攻击只在特定相位可见或造成伤害 |
| **风格排斥** | 无。在这场最终的战斗中，任何策略都是被允许的，只要你能活下来 |

#### 战斗流程：频谱的四种形态

| Boss 形态 | 攻击模式 | 玩家应对 |
| :--- | :--- | :--- |
| **正弦波 (Sine)** | Boss 释放平滑的正弦波干涉图样，只在**低通相位**可见并造成伤害 | 切换到低通相位观察波形并躲避，或切换到其他相位使其无效 |
| **方波 (Square)** | Boss 生成大范围的矩形致死区域，只在**全频相位**生效 | 避免停留在全频相位，或在致死区域生成前快速离开 |
| **锯齿波 (Sawtooth)** | Boss 释放密集的、快速的锯齿状弹幕，只在**高通相位**可见并造成伤害 | 切换到高通相位观察弹幕轨迹并躲避，或切换到其他相位使其无效 |
| **白噪音 (Noise)** | Boss 进入白噪音形态，全屏持续伤害，在所有相位都生效 | 这是唯一的强制输出窗口。必须切换回**全频相位**，攻击 Boss 暴露的核心，才能打断白噪音形态。长时间停留在极端相位会导致频谱偏移疲劳过高而无法有效输出 |

#### 最终阶段：频谱崩溃

当 Boss HP 低于 10% 时，它会进入"频谱崩溃"状态。所有四种波形的攻击将同时出现，屏幕被海量的弹幕、致死区域和视觉噪音淹没。此时，玩家唯一的生路是释放最强大的**减十三和弦（终焉乐章）**。其长达 4 拍的延迟施法时间，在这片混乱中显得无比漫长和艰难。成功释放后，游戏将进入终焉过场动画，玩家获得了最终的胜利。

## 8. 总结与展望

本文档 v3.0 在 v2.0 的宏观叙事框架下，为《Project Harmony》的七个核心章节提供了具体、可执行的波次级关卡设计方案。设计严格遵循"环境即教程"、"先安全试错再实战检验"和"辩证否定链"三大原则，旨在通过"恰到好处的引导"，让玩家在不断解决新问题的过程中，循序渐进地、自然地掌握游戏的所有核心机制——从基础的节奏感到复杂的频谱相位相位切换。

每一章的设计都以前一章的教学成果为基础，并引入新的核心矛盾，迫使玩家打破已有的舒适区，探索新的策略组合。Boss 战作为每章的"期末考试"，不仅在机制上综合考验玩家的掌握程度，更在音乐风格和叙事上与章节主题深度融合，最终构建出一条从古典秩序到现代噪音，从谱写到即兴，从音乐到混音的完整成长路径。

本设计文档为后续的开发、测试和平衡性调整提供了坚实的蓝图。未来的工作应聚焦于：

1.  **原型实现与测试**：尽快将本文档的设计转化为可玩原型，通过玩家测试验证引导的有效性和难度曲线的合理性。
2.  **数值微调**：基于测试反馈，利用自动化跑分系统 [2] 对敌人 HP、伤害、波次密度、疲劳阈值等核心数值进行精细调整。
3.  **视觉与音效**：为每个章节、每种敌人、每种相位形态设计独特的视觉风格和音效，强化机制的可识别性和艺术表现力。

最终，我们期望通过这套精心设计的关卡，将《Project Harmony》打造为一款不仅在机制上富有深度，更能在情感和智性上引导玩家体验"音乐性"本身的游戏杰作。
