# Project Harmony - 游戏设计文档

## 项目简介

**Project Harmony** 是一款将音乐理论与魔法系统深度结合的幸存者类（Survivor-like）肉鸽（Roguelike）游戏。本仓库包含了该游戏的完整设计文档（GDD），记录了从初始创意会议中提取的所有核心机制、设计灵感和待定议题。

## 核心创新点

游戏的核心创新在于将玩家的战斗行为转化为音乐创作过程：

- **音乐即魔法：** 所有法术都基于音乐的音符、和弦和节奏构建
- **听感即平衡：** 利用人类对和谐与不和谐的直觉感受来构建游戏平衡机制
- **战斗即创作：** 每一轮战斗都是一次从噪音到乐章的音乐创作之旅

## 文档结构

- **GDD.md** - 完整的游戏设计文档，包含：
  - 游戏概述与设计支柱
  - 核心游戏机制（节奏同步战斗、听感疲劳系统、和谐与不和谐系统）
  - 法术与成长体系（十二平均律、白键/黑键映射、和弦系统）
  - 扩展性设计（基于调性的角色系统）
  - 后续议题与待办事项
