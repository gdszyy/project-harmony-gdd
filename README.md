## 项目简介

**Project Harmony** 是一款将音乐理论与魔法系统深度结合的幸存者类（Survivor-like）肉鸽（Roguelike）游戏。本仓库是该游戏的设计文档库，所有设计决策与迭代过程均在此记录。

## 文档结构

- **`GDD.md`**: **[核心文档]** 这是游戏的**实时主设计文档 (Live GDD)**，包含了当前最新、最完整的游戏设计方案。所有最新的机制、参数和系统都在这里定义。

- **`Assets/`**: 存放当前 GDD 中引用的所有最新可视化图表和资源。

- **`Archive/`**: 存放所有历史版本的设计方案文档和相关图表，作为设计迭代的记录存档。

## 当前设计核心 (v5.0)

当前的设计方案构建了一个完全由参数化和乐理规则驱动的生成式法术系统：

1.  **白键法术**：作为基础弹体的四维参数空间。
2.  **黑键修饰符**：作为弹体行为的即时修饰，或作为和弦构成音。
3.  **和弦类型**：决定法术的**形态**（投掷物、爆炸、DOT、法阵、冲击波、区域、召唤等）。
4.  **和弦进行**：基于**和弦功能转换**的通用规则触发乐段效果。
5.  **节奏型**：改变弹体的行为模式。

详细内容请查阅 **`GDD.md`**。
